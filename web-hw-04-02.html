Аккордеон

Нажмите на заголовок, чтобы открыть или закрыть текст. Одновременно открыт только один пункт.

HTML — это язык разметки, с помощью которого создаётся структура веб-страницы.

CSS используется для оформления страницы: цвета, отступов, размеров и расположения элементов.

JavaScript добавляет интерактивность: кнопки, игры, формы, меню и другие действия на странице.
